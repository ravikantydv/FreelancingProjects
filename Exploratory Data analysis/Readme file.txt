Requirements for runnning the notebook--
Python with  3.11.2 version should be installed in your system,

You should have installed libraries in your system like--
 Pandas, 
 Numpy, 
 scikit-learn,
 sklearn,
 mlxtend,
 etc..


After unzipping the folder, you have a .ipynb file, upload it to collab or anaconda and load the dataset to your notebook then you can run the cells.
There is another file that is explaining the insights from visualization with code in python. 

If you want to run .ipynb file into vscode you can also use it there by installing the extension according to your needs.
